# Restaurant On-Line

This is a demo Web Application for ordering meals by internet - Restaurant On-line.

# Tools and technologies used
  * Spring Boot - 2.1.8.RELEASE
  * JDK 11
  * Spring Framework - 5.1.9.RELEASE (Spring MVC)
  * Maven - 3.6.2
  * IDE - Eclipse with STS
  * H2
  * Thymeleaf

# Instalation and run
 - clone repository
 - mvn install
 - mvn spring-boot:run
